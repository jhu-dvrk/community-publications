# dVRK Publications Website

Vue.js website for browsing and searching dVRK (da Vinci Research Kit) publications.

## Quick Start

Run the website with a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit: `http://localhost:8000`
